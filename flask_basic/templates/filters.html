More built in filters:

  1. Set default value: {{ name|default('Name is not defined', True) }}

  2. Make a list: {{ film|list }}

  3. Make replace: {{ film|replace("Christmas", "Wedding")|upper }}

  4. First item from dict: {{ table|first }}
